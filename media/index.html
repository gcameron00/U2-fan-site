Adding Your Own Photos

To replace these placeholders with real photos:

  1. Create a folder: assets/images/
  2. Add your image files (JPG, WebP recommended)
  3. Replace the .media-placeholder div with an <img> tag
  4. Add data-lightbox="assets/images/your-file.jpg" to the .media-item div to enable the lightbox
